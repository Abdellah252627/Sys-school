كيفية تثبيت النظام

متطلبات النظام
  • Node.js إصدار 14 أو أحدث
  • npm أو yarn
  • SQLite (مضمن مع النظام)
خطوات التثبيت
# استنساخ المستودع
git clone https://github.com/Abdellah252627/Sys-school.git

# الدخول للمجلد
cd Sys-school

# تثبيت المكتبات
npm install

# تشغيل النظام
npm start

سيتم تشغيل النظام على المنفذ 3000: http://localhost:3000
